Design a cute, warm, and approachable web app UI for an AI-powered Alzheimer’s prediction tool. The interface should feel comforting, human, and slightly playful, not clinical.

💖 Visual Style
“Soft cute” aesthetic (think cozy, gentle, friendly)
Rounded EVERYTHING (buttons, cards, containers)
Slightly playful but still respectful
Inspired by:
Health apps + cozy illustration styles
Soft pastel UI (not childish, but warm)
🎨 Color Palette
Background: warm off-white / cream
Primary: soft pastel blue
Secondary: lavender or dusty purple
Accent: soft sage green
Highlight: peach/light coral (for buttons or alerts)
Text: dark gray (soft, not harsh)
🧓 Character & Icon Style
Include tiny grandparent-style characters/icons
Simple line illustrations
Smiling elderly faces, glasses, soft hair
Can sit on cards or peek from corners
Style:
Minimal
Rounded
Friendly (no realism)
🧩 Layout Structure
🏠 Home Screen
1. Header
Title: App name (rounded, friendly font)
Add a tiny grandparent icon waving 👋
Right side:
Menu icon
Profile/settings icon (rounded)

Big rounded card
Text: “Upload your data”
Add:
Cute grandparent holding a file OR
Cloud with a smiling face
Button:

Soft pastel color

Rounded dropdown or pill buttons
Label: “Choose model”
Options:
CatBoost

Random Forest

4. Results Section
Large soft card labeled “Results”
Add:
Friendly grandparent avatar reacting (happy/neutral)

Prediction result
Confidence score (styled as a soft progress bar)
5. Stats & Graphs
Two side-by-side rounded cards:
“Stats”
“Graphs”
Add tiny decorative icons:
📊 but in soft/custom style

6. Info Section (Bottom)
Two soft cards:
“Early Warning Signs”

Add small comforting icons (heart, brain, smile)
📄 About Page
Same soft theme
Sections:
“How it works”

Add a grandparent reading a book 📖 in corner

Buttons slightly “pop” on hover

Characters can subtly react:
Smile on success
Neutral on loading

Rounded sans-serif (e.g.:
Nunito

)

Reassuring, not alarming
Friendly language:
